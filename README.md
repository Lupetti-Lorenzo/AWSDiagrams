# AWSDiagrams
Cloud Diagrams with python diagrams library

## How to start - Diagrams py

### Install graphviz (Mac)

```bash
brew install graphviz  
```

### Setup python

```bash
pipenv shell
pipenv install
```

## Generate the diagrams

```bash
python FILENAME.py
```
